What is the effective rate of return?
The effective rate of return is the rate of return that is generated by a specific investment when all factors impacting receipts are considered. This approach is the most comprehensive view of the return on an investment.
How to calculate effective rate of return?
To calculate effective rate of return you should use the following formula:
effective rate of return = (1 + i / n)^n - 1
where:
n = number of periods
i = nominal interest rate
Why use effective rate of return?
The effective rate of return is usually used by borrowers to understand the true cost of a loan and the investors use it to get the actual expected return on an investment.
